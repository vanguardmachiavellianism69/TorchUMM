## 🧭 What TorchUMM Does

TorchUMM is a unified multimodal model toolkit for Windows users who want one place to work with text, images, audio, and other model inputs.

Use it to:
- load and manage multimodal models
- run local model tasks in one app
- switch between inputs without extra setup
- keep your work in a single tool
- use a clean interface that is easy to follow

## 💻 What You Need

TorchUMM works best on a Windows PC with:
- Windows 10 or Windows 11
- 8 GB RAM or more
- 5 GB free disk space
- a modern Intel or AMD processor
- a stable internet connection for the first download

For larger models, more RAM and more free disk space can help.

## ❓ Help With Common Issues

### The app does not open
- Check that the download finished
- Make sure you extracted the ZIP file if needed
- Try running the app as an administrator
- Confirm that Windows did not block the file

### The app opens, then closes
- Restart the app
- Re-download the file if it looks damaged
- Check that your PC has enough free RAM
- Remove and reinstall the app in a fresh folder

### Models do not load
- Check the model file path
- Make sure the model file is fully downloaded
- Use a folder with enough free space
- Keep file names simple

### The screen looks blank
- Wait a few seconds for the app to finish loading
- Resize the window
- Restart TorchUMM
- Reinstall if the issue keeps happening
